This is a batch file so that you can preview the .udn source file you are translating.
This instruction assumes that you use Araxis Merge as the tool for the translation.

1. Copy "nConvert to HTML.bat" C:\Users\<UserName>\AppData\Roaming\Microsoft\Windows\SendTo.
2. Click [...] button right beside the filename field.
3. Right click on the INT.udn or XXX.udn > Send to (N) > nConvert to HTML (RMB - N - N).
